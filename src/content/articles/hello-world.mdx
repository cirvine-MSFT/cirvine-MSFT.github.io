Welcome to the site. I'm Casey, and this is where I'll be sharing what I'm building, learning, and tinkering with.

## What to Expect

- **Short-form videos** (5–15 min) on topics like AI-assisted development, developer tools, and whatever I'm currently rabbit-holing on
- **Blog posts** that go deeper than a video can, with code samples and links
- **Project write-ups** for the things I'm building in public

## How This Site Works

This site is built with [Astro](https://astro.build), styled with [Tailwind CSS](https://tailwindcss.com), and hosted for free on [GitHub Pages](https://pages.github.com). Every article is an MDX file in a git repo. Push to `main` and it deploys automatically.

The whole thing was scaffolded in one session with GitHub Copilot CLI. The source is open: [cirvine-MSFT/cirvine-MSFT.github.io](https://github.com/cirvine-MSFT/cirvine-MSFT.github.io).
