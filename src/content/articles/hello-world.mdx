# Hello World

This is a draft post to test the site scaffolding. Set `draft: false` in the frontmatter to publish it.

## YouTube Embeds

To embed a YouTube video in any article, add `youtubeId` to the frontmatter:

```yaml
youtubeId: "dQw4w9WgXcQ"
```

The video will appear at the top of the article automatically.

## Writing Articles

1. Create a new `.mdx` file in `src/content/articles/`
2. Add frontmatter (title, description, pubDate, tags)
3. Set `draft: true` while working on it
4. Preview locally with `npm run dev`
5. Set `draft: false` and push to publish
